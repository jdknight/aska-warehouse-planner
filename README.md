# ASKA Warehouse Planner

Provides a [ASKA][aska] Warehouse template drawing supported by
[draw.io/diagrams.net][drawio]. Aims to provide a quick way for users to
plan out their warehouses before building them in the game.

## Instructions

1. Open the [application][drawio].
2. Select `File -> Open from -> URL...`.
3. Enter the following URL:

   ```
   https://raw.githubusercontent.com/jdknight/aska-warehouse-planner/main/template/AskaWarehouseTemplate.drawio
   ```

Users may also opt to downloading the [template][template] file manually and
opening it in their application.


[aska]: https://playaska.com/
[drawio]: https://app.diagrams.net/
[template]: https://github.com/jdknight/aska-warehouse-planner/tree/main/template
